# Todo App

軽量なWebベースのTodoアプリ（HTML/CSS/JS）。localStorageでデータを保存します。

使い方:

- 開く: `index.html` をブラウザで開きます。
- タスク追加/編集/削除、サブタスク、期限、繰り返し、優先度、カテゴリ、タグ、添付ファイルをサポートします。

GitHub Pages に公開する手順:

1. リポジトリを作成し、このフォルダのファイルをコミットします。
2. `gh-pages` ブランチに `index.html` と他のファイルを置くか、リポジトリの `main` ブランチの `gh-pages` 設定で `root` を指定します。
3. GitHub リポジトリの Settings → Pages で公開を有効にします。

注意:
- データはブラウザの localStorage に保存されます。別ブラウザやデバイス間で同期されません。
